NBA Betting Model

I've been an avid fan of the NBA for a while, and I consistently watch multiple games per week during the season. I have attempted to sports bet with limited success, so I wondered if it would be possible to make a model that could accurately predict scores of upcoming games. This was unlike any project I'd done before, so I was able to learn and implement new technologies as I went along.


Skills and Technologies Used

  • Python
  • BeautifulSoup
  • Selenium
  • Pandas
  • Scikit-learn
  • Git

Project Details

To collect historical data, I chose to scrape historical box scores from basketball reference. I was able to find scrape urls for every box score for each season using BeautifulSoup, and continued by scraping all team stats from each of those games. Then, I cleaned up the data by removing columns irrelevant to team stats, such as game score or +/-, as well as removing empty rows from my dataset.


Next, I had to collect historical betting data. I found this to be more difficult, as sites such as DraftKings don't make it easy to access historical data. I found a website where these historical spreads and totals were hidden behind some javascript, so I wasn't able to use BeautifulSoup like I did with the box scores. Instead, I used selenium to iterate through every day in the NBA season, open the url in an automated browser, and pull the DraftKings odds for each game on that day. Then, using Pandas, I matched each game's odds with the box score data I had for the same game.


Now I had to prepare my dataset for training. I decided to use rolling averages as the input for each game, as with predicting future matchups I only have data from before the game. So I prepared my dataset on a 10-game rolling average for each stat and advanced stat.


Outcome

With scikit-learn, I tested out many different training models, and settled on RandomForestRegression, as it gave me the highest spread and totals accuracy of 51% and 52% respectively. I am excited to test this model with the upcoming NBA season.


Next Steps

Despite having a working model that predicts over 50% correct, there are more features that I want to add. Injury reports, rest day advantage, and individual player stats are all things I believe can greatly improve the accuracy of my model, but will require time to implement. Nonetheless, I have enjoyed this project so far, and motivated to continue improving it.

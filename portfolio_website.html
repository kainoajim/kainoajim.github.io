Project Details

The most difficult aspect of this project was the CSS. I didn't particulary love front-end UI development at the time I started this project, but I figured it was something that could be improved. While this website is simple, styling it was not the easiest thing to do.

I really enjoyed working with JavaScript to add visible effects to the page, such as changing the theme from light to dark, as well as adding some effects on the buttons.


Outcome

I will continue to work on this website, as there is a lot that can be improved, but I am happy with the result of a readable, clean design.
